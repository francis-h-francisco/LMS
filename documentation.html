Course Management Guide

Learn how to manually create, edit, and manage courses using JSON files

üìÅ File Structure Overview

data/courses.json - Main course catalog
data/course-{id}.json - Individual course content
üí° Pro Tip: You can use either the visual Course Builder or manual JSON editing - both methods work together!

üÜï Adding a New Course

1 Update courses.json

Add your new course to the courses array in data/courses.json:

{ "courses": [ { "id": "your-course-id", "title": "Your Course Title", "description": "Course description here", "image": "https://example.com/image.jpg", "dataFile": "data/course-your-course-id.json", "certificateTitle": "Certificate Title", "certificateMessage": "Certificate message here" } // ... other courses ] }

2 Create Course Content File

Create a new JSON file following the naming pattern course-{id}.json in the data/ folder:

{ "courseTitle": "Your Course Title", "modules": [ { "moduleTitle": "Module 1 Title", "lessons": [ { "id": "m1l1", "title": "Lesson 1.1 Title", "content": [ // Content items go here ] } ] } ], "quiz": { "id": "quiz", "title": "Final Assessment", "questions": [ // Quiz questions go here ] } }

‚úèÔ∏è Editing an Existing Course

1 Update Course Information

Edit the course entry in data/courses.json to change title, description, image, or certificate details.

2 Modify Course Content

Edit the corresponding course-{id}.json file to update modules, lessons, or quiz questions.

‚ö†Ô∏è Important: If you change a course ID, make sure to also rename the corresponding JSON file and update the dataFile reference in courses.json.

üóëÔ∏è Deleting a Course

1 Remove from courses.json

Delete the course entry from the courses array in data/courses.json.

2 Delete Content File

Remove the corresponding course-{id}.json file from the data/ folder.

‚ö†Ô∏è Note: This will not delete user progress data. User completion records will remain in the system but the course will no longer be accessible.

üîß Troubleshooting

Common Issues

Course not appearing?
Check that the course ID in courses.json matches the filename pattern: course-{id}.json
JSON syntax errors?
Use a JSON validator tool. Ensure all quotes, commas, and brackets are properly closed.
Images not loading?
Verify image URLs are accessible and use HTTPS if possible.

Validation Tools

  • Use JSON Formatter to validate your JSON
  • Check browser console for JavaScript errors (F12)
  • Test image URLs directly in your browser
